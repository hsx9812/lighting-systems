## FAQ 常见问题

**如何选择最合适的LED灯具？**
根据场景需求（如亮度、色温、显色性）与空间面积，结合功率和配光曲线进行对比，再参考用户评价快速筛选。

**智能照明系统部署难吗？**
多数平台提供即插即用的Wi-Fi/蓝牙模块，配合手机App或语音助手，只需几分钟完成配网与场景设置。

**如何评估照明项目成本？**
可参考平台提供的批量优惠和工程采购报价，结合耗电量与灯具寿命计算全生命周期成本。
